(a) How does javascript work?

ANS: Javascript is an interpreted, single-threaded language. It will execute one line of code at a time. In order to preview the code, it parses HTML, CSS, and JS code. Then it renders that code using javascript. Once all the code has been executed, it runs the code on the browser and displays a preview of the webpage.

(b) What does event loop do?

ANS: Javascript runs code one at a time. Whenever a function appears in the code, it will read it. By sending the code to the Browser Task queue, the javascript system can run its next code without being blocked. When the queue task has been processed completely, the code is sent to the browser preview.

(c) What is the difference between LocalStorage and SectionStorage.

ANS: LocalStorage is accessible from any browser window. However, SessionStorage is only accessible by one/the same window. LocalStorage data is stored forever. You can manually delete LocalStorage data. SessionStorage data expires/is removed after a tab/window is closed.
